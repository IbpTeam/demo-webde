Inputer库

Inputer库用于桌面上右键–>rename显示的文本输入框，用于文件的重命名。

使用方法


  var _inputer = Inputer.create('name');
  _options = {
  'left': 100,
  'top' : 100,
  'width': 100,
  'height': 100,
  'oldtext': 'text',                 //用于初始显示时，显示在输入框的文字。
  'callback': function(newtext){   //newtext输入框输入的文字，返回的文字。
          ....
    }
  }
  _inputer.show(_options);
