Scroller

scroller是一个滚动条控件，用以实现跨浏览器的滚动条控件的实现，其使用的控件是nanoScrollerjs，地址为：nanoScroller

scroller是对nanoscroller的封装，其使用方式如下：

使用方法


  <div class='nano'>
  <div class='nano-content'>
  </div>
  </div>
  使用封装方法：
	var scroller = Scroller.create(options);   //options为设置选项，也可以使用scroller.setOptions来进行设置。
        scroller.attach($('.nano'));
	或scroller.refresh($('.nano')); 
  使用原jquery方法
       $('.nano').nanoScroller();
详细使用方法参考原网站：nanoScroller

设置选项


  scroll:top/bottom       //滚动条位置
  scrollTop/scrollBottom: value      //滚动条偏移位置
  scrollTo:$(DOM)      //滚动到某一元素位置
  stop:true         //停止滚动事件
  destory:true        //恢复浏览器自动滚动事件
  flash: true            //自动刷新。
