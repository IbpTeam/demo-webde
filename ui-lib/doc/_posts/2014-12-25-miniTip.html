MiniTip

MiniTip是悬浮提示框，可在任意元素上添加使用

使用方法


var tip = MiniTip.create('anyElementId', {
  content: 'hello, miniTip', 
  event: 'hover'
});
其中第一个参数是欲添加MiniTip的元素的id（注意不是MiniTip的id），
第二个是指定的选项，更多请参考下面内容

可以指定的选项


      title                -------------        标题，不指定则没有标题栏
      content          -------------        内容
      delay             -------------        提示框出现前和隐藏前的延迟（单位ms）
      anchor           -------------       位置，n (top), s (bottom), e (right), w (left)
      event             -------------        呈现提示框的触发事件，可以为'hover' , 'click' 或者 'custom'（自定义）
      fadeIn           -------------        淡入延迟（单位ms）
      fadeOut         -------------        淡出延迟（单位ms）
      aHide            -------------         将鼠标移至提示框，提示框是否消失（默认消失，欲设置不消失将此值设为false）
      maxW           -------------         提示框的最大宽度
      offset            -------------         离添加提示框的元素距离（默认5px）

提供的接口

当event参数指定为'custom'，则提示框的呈现和隐藏由程序员控制，提供两个接口:


  1. show    呈现
  2. hide      隐藏

demo

示例1. 点击触发

click on me! 


MiniTip.create('tip1', {
  event: 'click',
  content: '示例1. 点击触发'
});

示例2. 停靠触发

Hover over me! 


MiniTip.create('tip2', {
  title: '示例2',
  event: 'hover',
  content: '停靠触发',
  delay: 1000,
  anchor: 'e'
});

示例三. 用户自定义触发


var tip3 = MiniTip.create('sendid', {
    event: 'custom',
    content: '发送内容不能为空！'
});

function clicksend(){
  if (document.getElementById('textid').value !== ''){
    document.getElementById('textid').value = '';
  }
  else {
    tip3.show({
      content: '发送内容不能为空！' + count++  //此处若为tip3.show()，则显示样式和内容不变

}
