modalBox

modalBox可用于生成模态对话框。


使用方法：



  ModalBox.create($('.modalBox'),{   //$('.modalBox'')是要放在modalbox的jquery对象，可以是div，内部包含任意内容
    iconImg:'images/x.png',      //关闭按钮的图片
    iconClose:true,                     //是否使用图片关闭
    keyClose:true,                      //是否使用键盘关闭
    bodyClose:true                    //是否使用点击空白处关闭
  });

配置选项

下面是可配置的选项，配置配置选项，默认选项如下：


  ModalBox._options = {
    width: 'auto',                //宽度，默认为自动
    height: 'auto',              //高度，默认为自动
    left: 'auto',                   //位置左，默认位置居中
    top: 'auto',                  //位置上，默认居中
    overlay: true,             //全屏底幕
    iconClose: false,        //是否使用图片关闭
    keyClose: true,          //是否使用键盘关闭
    bodyClose: true,       //是否使用点击空白处关闭
    iconImg: 'img/close.png',    //关闭按钮图片
    onOpen: function () {},         //开启modalBox时，执行的方法
    onClose: function () {}          //关闭modalBox时，执行的方法
  };
