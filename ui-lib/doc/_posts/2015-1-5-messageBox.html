MessageBox

MessageBox是消息对话框，屏蔽其他元素，让用户必须进行当前对话

使用方法


  var msgBox = MessageBox.create('testid', {
    title: '',
    width: 350,
    height: 200,
    close: true
  });
  msgBox.setButton([
      {text:'确  定', clkaction: function(){alert('hello1'); msgBox.close();}},
      {text:'取  消', clkaction: function(){alert('hello2')}}
    ]);
  msgBox.append('<p style="font-size:25px; text-align: center;"><b>确定删除该文件吗？</b></p>');

提供的接口


  1. setButton    设置按钮
  2. append        在对话框中添加内容
  3. close             关闭
