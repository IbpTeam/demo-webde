MessageBox

MessageBox是消息对话框，屏蔽其他元素，让用户必须进行当前对话

使用方法


var msgBox = MessageBox.create('testid', {
  title: '',
  width: 350,
  height: 200,
  close: true,
  buttons: [{
    text: '确  定',
    clkaction: function() {
      alert('hello1');
      msgBox.hide();
    }
  }, {
    text: '取  消',
    clkaction: function() {
      alert('hello2');
    }
  }]
});

function clickDemo() {
  msgBox.post('确定删除改文件吗?');
}

提供的接口


  1. post              在对话框中添加内容
  2. close             关闭
  3. hide              隐藏
