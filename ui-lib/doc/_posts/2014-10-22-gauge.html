Gauge

Gauge库提供了简单的接口生成进度条控件，该进度条控件具有多种色彩以及多种功能，可以提供丰富的渲染效果，而且操作简单。具体使用方法见下文

使用方法


  var _gauge = Gauge.create(options);   //创建gauge对象
  _gauge.add($('#parentID'));           //向parent中添加_gauge

提供的接口

此_gauge对象提供以下接口：


  _gauge.add($('#parentID'));           //向parent中添加_gauge
  _gauge.setOptions(options);           //设置新属性
  _gauge.modify(self，options);         //设置进度条元素self新的属性并更新显示新属性。

option说明

本gauge提供了丰富的设置属性，下面对整个可设置属性进行详细说明。


this._options = {
  width: 400,        //宽度 取值范围（48～n）
  height: 30,        //高度 取值范围（12～n）
  reflex: 0.5,       //倒影的透明度 取值范围（0.001～1）
  opacity: 0.25,     //空白区域的渐变属性 当limit：true时生效 取值范围（0.001～1）
  pulse: 100,        //脉动速度，颜色渐变速度（毫秒级） （30-1000） 
  limit: false,      //有最大值模式，values的最有一位数字是该进度条的最大值
  noscale: false,    //是否没有刻度，
  gradient: false,   //颜色渐变（左右渐变）limit:true 和 values只有两个数值时生效
  vertical: false,   //横向和纵向 默认横向
  busy: false,       //颜色脉动变色
  radius: 1.0,       //边角的圆角度 取值范围（0.001~1）
  scale: null,       //刻度格长 null为自动 取值范围（2～n）
  empty: '#cccccc',  //空白区域颜色
  name: null,        //名字，id
  values: null,      //数值数组用于设置进度条的显示长度， 数组格式
  colors: ['#3765d9','#9ede7c','#9e42ee','#ec7612','#00aaaa','#cc0000','#aaaa00','#008000'] //各个数值的颜色
};
