Unslider是一款图片轮播插件，该插件具有操作简单，使用方便的特点

使用方法

建立图片轮播的方法有两个步骤:
第一步：定义图片结构，
第二步：应用轮播 

  
  	第一步：定义图片结构，
  <div class="banner">
    <ul>
    <li style="background-image: url('/assets/img/shop.jpg')">
      这是一张在路上的照片
    </li>
    <li style="background-image: url('/assets/img/subway.jpg')">
      这是一张地铁的照片
    </li>
    <li  style="background-image: url('/assets/img/sunset.jpg')">
      这是一张太阳落山的照片
    </li>
    </ul>
  </div>
	

	第二步：应用轮播，
	$('.banner').unslider(options);  //设置内容将在下面介绍。

设置属性

  
  $('.banner').unslider({
	speed: 500,               // 轮播动画速度
	delay: 3000,              // 轮播延时
	complete: function() {},  // 完成轮播时的响应函数
	keys: true,               // 支持左右按键转换图片
	dots: true,               // 显示图片上第几个点
	fluid: false              // 支持响应设计，屏幕变化
});

操控unslider

可以在轮播时操控该轮播插件，见如下：

  
  var slidey = $('.banner').unslider(),
    data = slidey.data('unslider');
    data.play();                //开始轮播
    data.stop();                //停止轮播
    data.to(2, function() {});  //图片跳转到第2张，并执行函数，从0开始计数
    data.keys();                //支持键盘
    data.next();                //下一张图片
    data.prev();                //上一张图片
    data.dots();                //显示dot
